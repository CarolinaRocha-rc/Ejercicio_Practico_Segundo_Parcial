# Ejercicio_Practico_Segundo_Parcial
En este repositorio se encontrará el ejercicio propuesto en el examen parctico del Segundo Parcial
Aplicación desarrollada en C++ como parte de la asignatura Algoritmos y Lógica de Programación de la Universidad Técnica de Ambato.

El sistema permite realizar operaciones matemáticas básicas, registrar notas de estudiantes, calcular estadísticas académicas y guardar resultados en archivos de texto.

Características
  Menú interactivo en consola.
  Operaciones matemáticas básicas:
    Suma
    Resta
    Multiplicación
    División con validación para cero
Registro de estudiantes y notas.
Cálculo de:
Promedio
Nota mayor
Nota menor
Cantidad de aprobados
Cantidad de reprobados
Almacenamiento de resultados en archivo .txt.
Validación de datos ingresados.
Tecnologías utilizadas
Lenguaje: C++
Editor: Visual Studio Code
Compilador: g++
